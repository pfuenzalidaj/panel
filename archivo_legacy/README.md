# Archivo histórico

Contenido conservado para trazabilidad y recuperación.

- `implementacion_v2_1/`: componentes activos antes de la estandarización 2.2.0.
- `fuentes_originales/`: archivos recibidos con sus nombres originales.
- `temporales_recuperados/`: copias temporales recuperadas de sesiones anteriores.

Estos archivos no participan en la ejecución del entregable vigente.
La versión activa se encuentra en `../Dashboard_Ejecutivo_TI/`.
